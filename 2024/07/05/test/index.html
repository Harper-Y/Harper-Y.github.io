测试文章

前言：基于Windows11系统。hexo是一个基于Node.js的博客框架，与GitHub Pages搭配搭建轻量的个人网站。Hexo中有许多主题可供选择，考虑到维护以及个人审美，本文后续介绍基于butterfly主题的网站搭建。

ps:本文默认阅读者已拥有一个GitHub账户，且不对GitHub和Node.js做详细解释。

1.本地安装

  • git ：一个分布式版本管理工具

​ 官网：http://git-scm.com/downloads 【可能较慢】

​ 腾讯软件助手：https://pc.qq.com/detail/13/detail_22693.html【快一些，版本也是新的】

  • Node.js :一个开源的、跨平台的 JavaScript 运行时环境

    下载 | Node.js 中文网 (nodejs.cn)

这两个无需特殊设置，一路next就可以。C盘不够可以选路径时换位置安装。

  • 验证是否安装成功，在CMD依次输入：

    如果安装成功，每次会输出版本号（一些数字）

2.GitHub配置

  • 在本地某个盘中，创建一个文件夹Hexo（名字可以自定义，该文件夹将用来存储网站的相关信息）

    ，右键点入“Open Git Bash here” 得到一个命令行界面

然后退出进入CMD，输入 ssh-keygen -t rsa -C “你的 GitHub 邮箱”,回车至输出结束。

进入c:\用户\你的用户名 \ .ssh ；找到 id_rsa.pub 文件打开并复制

  • 进入GitHub，点击右上角进入settings，浏览左侧栏找到SSH and GPG keys，在最上侧SSH Keys，点击，把刚刚复制的id_rsa.pub的内容加入

  • 回到本地点击Hexo文件夹的Open Git Bash here，输入 ssh -T git@github.com。

    得到 ”are you sure you want to countining connect [yes/no…..] 输入 yes 然后回车

  • 显示 hi xxxxxxxxx…
